## ～ない

五段動詞のあ段未然形＋ない


| う     | あない    |     |
| ----- | ------ | --- |
| 行く    | 行かない   |     |
| 待つ    | 待たない   | まつ  |
|       |        |     |
| 見る    | 見ない    |     |
| 起きる   | 起きない   | おきる |
| 食べる   | 食べない   |     |
| 来（く）る | 来（こ）ない |     |
| する    | しない    |     |




## ～（ら）れる 可能态

五段動詞のあ段未然形＋れる　－　ｅる

xあれる会被约成eる

一段動詞の未然形＋られる

| う     | あ（ら）れる  |
| ----- | ------- |
| 行く    | 行ける     |
| 待つ    | 待てる     |
|       |         |
| 見る    | 見られる    |
| 起きる   | 起きられる   |
| 食べる   | 食べられる   |
|       |         |
| 来（く）る | 来（こ）られる |
| する    | できる     |

私は日本語が話せる。（話さえる　»　話せる）




## ～（ら）れる 被动态

五段動詞のあ段未然形＋れる
一段動詞の未然形＋られる

| う     | あ（ら）れる  |
| ----- | ------- |
| 行く    | 行かれる    |
| 待つ    | 待たれる    |
|       |         |
| 見る    | 見られる    |
| 起きる   | 起きられる   |
| 食べる   | 食べられる   |
|       |         |
| 来（く）る | 来（こ）られる |
| する    | される     |

私は先生に叱られる。
	叱る　しかる　骂

私わ、電車で隣の人に足を踏まれる。
	隣　となり
	足　あし
	踏む　ふむ

この橋は１０年前に建てられた。
	橋　はし
	前　まえ
	建てる　たてる

この作品は、魯迅（ロジン）によって書かれた。
	作品　さくひん
	書く　かく
	「によって」相当于「由～」

父に死なれた。




## ～（さ）せる　使役态

五段動詞のあ段未然形＋せる
一段動詞の未然形＋させる

| う     | あ（さ）せる  |
| ----- | ------- |
| 行く    | 行かせる    |
| 待つ    | 待たせる    |
|       |         |
| 見る    | 見させる    |
| 起きる   | 起きさせる   |
| 食べる   | 食べさせる   |
|       |         |
| 来（く）る | 来（こ）させる |
| する    | させる     |

父は、私**を**留学させた。（自動詞）

先生は、私**に**日本語を勉強させた。




## ～（よ）う　

表说话者的意志，或希望与听话者共同做什么。

五段動詞のお段未然形＋う
一段動詞の未然形＋よう

| う     | あ（さ）せる    |
| ----- | --------- |
| 行く    | 行こう　↘～    |
| 待つ    | 待とう　↘～    |
|       |           |
| 見る    | 見よう　↘～    |
| 起きる   | 起きよう　↘～   |
| 食べる   | 食べよう　↘～   |
|       |           |
| 来（く）る | 来（こ）よう　↘～ |
| する    | しよう　↘～    |
