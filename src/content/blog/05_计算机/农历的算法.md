# 阴历

初一即看不见月亮的一天，叫做朔，十五叫做望，是满月的日子。

![[Pasted image 20260314172436.png]]

朔望月的周期是29.53天。 若一年有12个月，则这样算出只有354.36天

但一个回归年有365.2422天，差了10.88天。

这10.88天可以使用19年补7个朔望月来消除误差。

一个月的29天/30天是不固定的，用于补偿0.53天。


# 阳历

二十四节气。节气严格以地球运动15度划分。

包括中气与节气，顺序交替进行，不含闰月的情况下，规定对应的阴历月必须包括相应中气。

![[Pasted image 20260314173516.png]]

当一个月找不到中气时，就称为闰月
